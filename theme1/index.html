Hummingbird is a simple, yet powerfull web framework. It is designed to be easy to use and is extremely flexible.

  • Minimal

    Hummingbird's core is lightweight; its power resides in plugins.

  • Simple

    Logical seperation of data and view logic ensures easy to maintain code.

  • Flexible

    A powerful plugin system leaves you free to use any ORM or templating engine you like.
